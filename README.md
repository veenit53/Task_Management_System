# 🚀 Task Management System (Frontend)

A modern and responsive **Task Management Web App** built using React, Vite, and Tailwind CSS.  
This application allows users to manage daily tasks efficiently with authentication and a clean UI.

---

## 🌐 Live Demo
👉 https://your-vercel-link.vercel.app

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 📋 Dashboard with all tasks
- ➕ Create new tasks
- 🔍 View task details
- 👤 User profile page
- 🔄 Client-side routing with React Router
- ⚡ Fast and optimized UI with Vite + Tailwind

---

## 🛠️ Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS
- React Router
- Context API (State Management)

**Backend (Connected)**
- Node.js
- Express.js
- MongoDB
